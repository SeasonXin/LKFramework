# 欢迎使用LKFramework框架

------

> * eclipse formatters
    本框架代码使用eclipse集成工具开发，定义了一些代码格式化。
> * lombok
    本框架中使用了lombok技术，提供了使用说明。
> * maven
    本框架使用了maven技术，且使用了自己的maven仓库，部分jar包在 https://mvnrepository.com/ 中是没有的。
